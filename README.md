# Bilheteria Inteligente

## Descrição

A Bilheteria Inteligente é uma Landing Page desenvolvida para apresentar uma solução de compra de ingressos online.

## Objetivo

Facilitar a compra de ingressos de forma rápida, segura e prática, reduzindo filas e melhorando a experiência dos usuários.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Layout responsivo
- Navegação entre seções
- Botão de ação
- Informações sobre o projeto
- Contato
